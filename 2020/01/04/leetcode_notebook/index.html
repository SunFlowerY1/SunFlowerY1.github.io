Is Subsequence

1、顺序查找法

explain:
如果 s[i] == t[j], 那么
对于 s[i + 1] 要从 t[j + 1] 开始查找.
code：
class Solution {
public boolean isSubsequence(String s, String t) {
char[] sCharArray = s.toCharArray();
char[] tCharArray = t.toCharArray();
int i = 0;
int j = 0;
while(i < s.length()){
while(j < t.length() && sCharArray[i] != tCharArray[j]){
j++;
}
if(j >= t.length()){
return false;
}
i++;
j++;
}
return true;
}
}

2、动态规划

explain:
要写矩阵，速度不如顺序查找
1、s(0) 与 t(0) 空串，匹配成功 dp[0][0] = true;
2、s(0) 与 t(j) j > 0, 匹配成功 dp[0][j] = true;
3、s(i) 与 t(j) i > 0, j > 0:
1) s[i - 1] == t[j - 1], dp[i][j] = dp[i - 1][j - 1];
2) s[i - 1] != t[j - 1], dp[i][j] = dp[i][j - 1]
code:
class Solution {
public boolean isSubsequence(String s, String t) {
if(s.length() == 0){
return true;
}
char[] sCharArray = s.toCharArray();
char[] tCharArray = t.toCharArray();
boolean[][] dp = new boolean[s.length() + 1][t.length() + 1];
for(int i = 0; i < t.length(); i++){
dp[0][i] = true;
}
int i = 1;
for(; i <= s.length(); i++){
for(int j = 1; j <= t.length(); j++){
if(sCharArray[i - 1] == tCharArray[j - 1]){
dp[i][j] = dp[i - 1][j - 1];
}else{
dp[i][j] = dp[i][j - 1];
}
}
}
return dp[s.length()][t.length()];
}
}
