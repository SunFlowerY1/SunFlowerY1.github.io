发表于 2020-06-05

一、安装文件下载（使用 rpm 方式安装）

 mysql-5.7.x
 rpm 文件列表：
   1、mysql-community-client-5.7.30-1.el7.x86_64.rpm;
   2、mysql-community-libs-5.7.30-1.el7.x86_64.rpm;
   3、mysql-community-common-5.7.30-1.el7.x86_64.rpm;
   4、mysql-community-server-5.7.30-1.el7.x86_64.rpm.

二、安装

  rpm -e mariadb-libs –nodeps (file 冲突解决)
  rpm -ivh mysql-community-common-5.7.30-1.el7.x86_64.rpm
  rpm -ivh mysql-community-libs-5.7.30-1.el7.x86_64.rpm
  rpm -ivh mysql-community-client-5.7.30-1.el7.x86_64.rpm
  rpm -ivh mysql-community-server-5.7.30-1.el7.x86_64.rpm

三、启动及配置

  systemctl start mysqld systemctl status mysqld
  获取初始密码: grep ‘temporary password’ /var/log/mysqld.log
  mysql -uroot -p
  设置密码策略和长度:
  use mysql; set global validate_password_policy=0;
  set global validate_password_length=1;
  重置密码:set password for root@localhost=password(‘123456’)
  配置远程登录:select user,host from user;
  show grants;
  grant all privileges on . TO ‘root’@’%’ identified by ‘123456’;
  flush privileges;
  时间默认值不能为 0 问题:select @@sql_mode;
  修改配置文件 /etc/my.cnf,添加
  sql-mode=ONLY_FULL_GROUP_BY,STRICT_TRANS_TABLES,
  ERROR_FOR_DIVISION_BY_ZERO,NO_AUTO_CREATE_USER,
  NO_ENGINE_SUBSTITUTION
  重启即可
